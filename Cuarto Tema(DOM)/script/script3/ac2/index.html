ACTIVIDAD 2

• Crea una página HTML con varios elementos (un párrafo, un encabezado y un botón). • Incluye una hoja de estilo en el documento con algunos estilos básicos. • Usa JavaScript para añadir dinámicamente nuevas reglas a esa hoja de estilo.
